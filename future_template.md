**Домен: управление устройствами**
- поддомен: система отопления
  - контекст: управление отоплением (включение/выключение системы отопления, установка температуры)
  - контекст: мониторинг температуры
- поддомен: управление освещением
    - контекст: включение/выключение освещения, автоматические сценарии управления освещением
- поддомен управление воротами
    - контекст: открытие/закрытие ворот, автоматический пропуск
- поддомен: видеонаблюдение
    - контекст: потоковое видео
    - контекст: хранение записей

**Домен: подключение устройств**
- поддомен: подключение устройств
    - контекст: установка устройств
    - контекст настройка устройств
    - контекст: удаление устройств

**Домен: менеджмент пользователей**
- поддомен: учетные записи пользователя
    - контекст: регистрация нового пользователя
    - контекст: управление ролями пользователя
    - контекст: удаление данных пользователя
- поддомен: подписки на сервисы
    - контекст: выбор сервисов
    - контекст: оплата подписки
    - контекст: продление подписки
    - контекст: отмена подписки

**Домен: поддержка пользователей**
- поддомен: обработка обращений пользователей
    - контекст: получение обращения
    - контекст: процессинг обращения
    - контекст: решение обращения
    - контекст: автоматические ответы на обращения
- поддомен: история запросов
    - контекст: хранение запросов
    - контекст: удаление старых запросов
